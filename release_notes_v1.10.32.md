## Mir2Ray v1.10.32

### Highlights
- Connection flow stability improvements and better connect/disconnect behavior.
- Real-delay based optimization logic for both **Give New Configs** and **Optimize**.
- Auto optimize + reconnect when ping is missing for 10s or goes above 600ms.
- Determinate progress bars for the first two action buttons.
- HEV native fallback handling to avoid crashes when native lib is unavailable.
- Branding updates: app name unified to Mir2Ray and new neon launcher icon.

### Artifacts
- Universal APK (recommended)
- ABI-specific APKs: arm64-v8a, armeabi-v7a, x86, x86_64
